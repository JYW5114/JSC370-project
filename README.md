# 🔥 Analysis of Factors Influencing Fire Incident Financial Loss in Toronto

## 🌐 Project Website

https://github.com/JYW5114/JSC370-project

## 📊 Project Overview

This project investigates the key factors that influence the financial loss caused by fire incidents in Toronto. Using data analysis and machine learning models, the goal is to understand how fire characteristics, response factors, and environmental conditions contribute to economic damage.

The project combines:

* Exploratory Data Analysis (EDA)
* Statistical Modeling and Machine learning
* Interactive visualizations

## 📁 Data Sources

### 1. Toronto Fire Incident Data

* Source: City of Toronto Open Data
* API Link: https://ckan0.cf.opendata.inter.prod-toronto.ca/api/3/action/
* Description: Contains detailed records of fire incidents, including response time, property type, fire severity, and estimated dollar loss etc.

### 2. Weather Data

* Source: Open-Meteo
* API Link: https://archive-api.open-meteo.com/v1/archive
* Description: Historical weather data (temperature, precipitation, snowfall) used to analyze environmental impacts on fire incidents.

## 📈 Key Features

* Analysis of fire incident patterns across time and location
* Investigation of response time and fire severity
* Modeling financial loss using machine learning
* Interactive dashboards and visualizations

## 🧠 Methods Used

* Data cleaning and preprocessing
* Feature engineering
* Regression models (log-transformed loss)
* Model evaluation (RMSE, MAE, R<sup>2</sup>)
* Visualization with Plotly and Dash
